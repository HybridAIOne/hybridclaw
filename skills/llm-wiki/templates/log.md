# Log

This is the append-only activity log for the wiki.

Append entries in reverse chronological order using headings like:

```md
## [2026-04-07] create | Wiki initialized
## [2026-04-07] ingest | Source Title
## [2026-04-07] archive | Superseded page
```

Each entry should state what changed, which pages were touched, and what
follow-ups remain.

Rotate this file when it becomes too long to scan comfortably.
